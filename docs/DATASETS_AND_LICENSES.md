# Dataset acquisition and terms

Dataset archives are not included in this repository. The benchmark downloads them into `replication_package/data_cache/` through Torchvision or scikit-learn.

## CIFAR-10

- Loader: `torchvision.datasets.CIFAR10`
- Source: <https://www.cs.toronto.edu/~kriz/cifar.html>
- Reference: Alex Krizhevsky, *Learning Multiple Layers of Features from Tiny Images* (2009).
- Redistribution: no archive is redistributed here; consult the official source for applicable terms.

## Fashion-MNIST

- Loader: `torchvision.datasets.FashionMNIST`
- Source: <https://github.com/zalandoresearch/fashion-mnist>
- Reference: Xiao, Rasul, and Vollgraf (2017), arXiv:1708.07747.
- Upstream repository license: MIT. The dataset is downloaded from the upstream mirrors and is not redistributed here.

## Kuzushiji-MNIST (KMNIST)

- Loader: `torchvision.datasets.KMNIST`
- Source: <https://codh.rois.ac.jp/kmnist/>
- Dataset DOI: <https://doi.org/10.20676/00000341>
- Upstream dataset license: CC BY-SA 4.0.

## 20 Newsgroups

- Loader: `sklearn.datasets.fetch_20newsgroups`
- Source: <https://archive.ics.uci.edu/dataset/113/twenty+newsgroups>
- Dataset DOI: <https://doi.org/10.24432/C5C323>
- UCI license: CC BY 4.0.
- Loader options: standard train/test subsets with headers, footers, and quotes removed.
- Implementation boundary: token sequences are truncated/padded to 96 content slots, no padding query/key mask is applied, positional embeddings are added to padded slots, and density uses all 97 slots including CLS.
